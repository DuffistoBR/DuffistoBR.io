O TECCO foi desenvolvido para controle de tratativa de eventos, por ser uma empresa de transporte a qualidade dos serviços é algo essêncial, pensando nisso realizei o desenvolvimento de um aplicativo que puxa atráves de um relatório do excel, todas as ocorrências diárias, após isso é enviado para a base de dados onde no aplicativo fica disponivel para o encarregado de cada operação realizar a tratativa dos eventos, atráves disso é possivel acompanhar o desempenho de cada operação e como estão sendo realizadas as tratativas em cada funcionário.
